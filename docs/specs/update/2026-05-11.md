# uniTerm 更新日志 - 2026-05-11

## 新功能

### 全局 UI
- **全新界面风格**：采用工业暗色主题重新设计全局 UI，电蓝色强调色搭配噪点纹理与层级阴影。同步提供配套浅色主题，支持一键切换，所有组件均适配双主题配色。

### AI 助手
- **模型切换器**：在 AI 侧边栏发送按钮左侧新增模型选择下拉菜单。切换模型时会同时更新全局配置中的默认模型（`settingsStore.activeModelId`）和当前 AI 调用的配置（`aiStore.config`）。
- **AI 存储 `setConfig()` 接口**：为 `aiStore` 新增 `setConfig(updates: Partial<AIConfig>)` 方法，支持程序化更新 AI 配置。
- **高危指令检测与三种执行模式**：
  - 自动识别高危指令（删除文件、关机、格式化、权限修改等）
  - **全部确认模式**（默认）：所有命令均需用户确认
  - **仅高危确认模式**：仅高危命令需确认，其他自动执行
  - **自动模式**：全部命令自动执行，无需确认
- **终端右键问 AI**：终端窗口右键菜单新增"问 AI"选项，仅在有选中文本时可用，点击后将选中内容发送到 AI 聊天框。

### 全局配置
- **设置页面**：新增设置按钮，点击后在主窗口区域打开设置标签页，采用左侧分类导航 + 右侧卡片式配置面板的左右布局，所有配置修改后直接生效。
- **基础设置**：主题设置（深色/浅色/跟随系统）、语言设置（简体中文/英文/系统默认）。
- **终端配置**：配色方案、字体、字体大小、选中后执行动作、右键功能、最大历史文本行数。
- **AI 助理设置**：
  - AI 模型配置改为列表模式，支持配置多个模型，可选择激活模型，支持修改和删除
  - 模型配置新增名称字段
  - AI 助理窗口发送按钮旁新增模型切换选项

## Bug 修复

### AI 助手
- **修复多工具调用 400 错误**：将 `pendingTool`（单对象）改为 `pendingTools`（FIFO 队列），正确处理跨轮次的多工具调用，避免产生格式错误的 API 请求，导致返回 HTTP 400 错误。
- **修复 LocalStorage 配额崩溃**：当 `localStorage` 存储配额超出时，增加自动截断和降级回退策略，防止应用因 `QuotaExceededError` 崩溃。
- **修复待处理工具自动拒绝逻辑**：当用户开始新对话时，所有待处理的工具调用都会被自动拒绝（而非仅拒绝第一个）。

### UI / 终端
- **修复终端输入卡顿**：解决每输入一个字符有近 1 秒延迟的问题，该问题在 AI 执行命令后会逐渐加剧。
- **修复多窗口创建卡顿**：同时创建多个终端窗口时界面不再出现明显卡顿。
- **修复多右键菜单冲突**：在不同位置点击右键不会再同时出现多个右键菜单，全局始终只保留一个右键菜单。
- **修复连接失败无提示**：连接失败的终端现在会显示失败原因提示，并支持回车键重连。
- **修复终端粘贴功能**：`pasteFromClipboard` 已正确标记为 `async`，以适配剪贴板 API 的异步调用。
- **修复 AI 侧边栏拖拽回弹**：修复 AI 区域调整大小时，拖动后约 1 秒突然回弹到之前宽度的问题。


## 优化

### AI 助手
- **会话存储压缩**：AI 会话持久化存储改用 `lz-string` UTF-16 压缩后再写入 `localStorage`，有效存储容量从约 5-10 MB 提升至约 15-50 MB。
- **消息保存限制上调**：
  - 每会话保留消息数：80 → **200**
  - 单条消息最大长度：3,000 → **10,000** 字符
  - 降级回退限制：20/500 → **50/2,000**
- **IN 框 UI 简化**：移除 IN 框头部的 `execute_command` 函数名标签，仅保留 "IN" 标签。
- **单命令执行限制**：Agent 循环强制每次 LLM 响应只处理一条 `execute_command` 工具调用，若模型返回多条仅执行第一条。
- **终端命令超时时间**：从 15 秒延长至 **60 秒**，超时后返回明确的超时提示信息。
- **AI 输入框优化**：输入框高度调高一倍，发送/模式按钮置于输入框内部右下方。
- **Markdown 渲染控制**：用户消息和调试消息不再使用 Markdown 格式渲染。
- **多标签页上下文感知**：每次给 AI 发送命令时自动携带当前标签页信息（标签页 ID、连接信息、会话类型），AI 能感知当前所在标签页。

### 连接边栏
- **移除过渡动画**：连接边栏动画速度与 AI 助手边栏保持一致，去掉过渡动画。
- **支持宽度调整**：连接窗口支持拖拽调整宽度。
- **单击/双击区分**：单击连接仅选中，双击才触发连接。
- **搜索功能**：支持通过名称和主机搜索连接。
- **展示连接账户**：连接列表除 IP 外，额外展示连接账户（`user@host:port`）。
- **按钮布局优化**：新建/编辑连接页面的按钮名称统一，"保存并连接"按钮置于最右侧。
- **名称自动填充**：名称字段选填，不填时自动使用主机作为名称，重名时自动增加后缀避免冲突。

### UI / 终端
- **标签文本居中**：标签栏中的标签标题水平居中对齐。
- **OUT 箭头对齐**：OUT 框展开箭头与 IN 框保持同一位置右对齐。
- **国际化完善**：新增大量 i18n 翻译键，覆盖 AI 助手、设置项描述、标签右键菜单、终端右键菜单、连接表单校验提示等（支持 zh-CN / en 双语）。
